## 1. 功能概述

认知模型可视化是AI Voice Interaction App的核心功能之一，通过直观的图形界面展示用户的认知模型和概念关系，帮助用户理解和优化自己的认知结构。

## 2. 技术栈

- **图形渲染**：SwiftUI Canvas + Core Graphics
- **图算法**：力导向图算法
- **动画效果**：SwiftUI内置动画
- **数据处理**：Combine框架

## 3. 可视化类型

### 3.1 力导向图
- **用途**：展示概念之间的关系强度和结构
- **特点**：节点之间的距离表示关系强度，边的粗细表示关系重要性
- **交互**：支持缩放、拖拽、点击查看详情

### 3.2 层次图
- **用途**：展示概念之间的层级关系
- **特点**：从上到下或从左到右的层次结构
- **交互**：支持展开/折叠、层级切换

### 3.3 雷达图
- **用途**：展示认知模型的多维度特征
- **特点**：多个维度的雷达图，直观展示各维度的强度
- **交互**：支持维度切换、数据对比

## 4. 核心组件

### 4.1 认知模型图组件
- **样式**：全屏图形容器，带有控制按钮
- **状态**：加载中、显示中、编辑中
- **交互**：缩放、拖拽、点击节点

### 4.2 概念节点组件
- **样式**：圆形或方形节点，带有概念名称和图标
- **状态**：默认、选中、高亮
- **交互**：点击查看详情，拖拽调整位置

### 4.3 关系边组件
- **样式**：连接两个节点的线条
- **状态**：默认、选中、高亮
- **交互**：点击查看关系详情

## 5. 功能流程

### 5.1 模型加载流程
1. 用户选择要查看的认知模型
2. 应用从后端获取认知模型数据
3. 解析数据并构建图结构
4. 使用力导向图算法计算节点位置
5. 渲染图形并显示给用户

### 5.2 模型交互流程
1. 用户点击节点查看详情
2. 显示节点的详细信息和相关关系
3. 用户可以编辑节点和关系
4. 保存修改并同步到后端

## 6. API集成

### 6.1 获取认知模型API
- **请求**：GET /api/v1/models/{modelId}
- **参数**：模型ID
- **响应**：认知模型的完整数据，包括概念和关系

### 6.2 更新认知模型API
- **请求**：PUT /api/v1/models/{modelId}
- **参数**：模型ID、更新的数据
- **响应**：更新后的认知模型数据

### 6.3 获取概念详情API
- **请求**：GET /api/v1/models/{modelId}/concepts/{conceptId}
- **参数**：概念ID
- **响应**：概念的详细信息

## 7. 错误处理

- **数据加载失败**：显示错误提示，允许用户重试
- **图形渲染失败**：显示简化视图，提示用户检查设备性能
- **交互操作失败**：显示操作失败提示，允许用户重试

## 8. 性能优化

- **数据分页**：对大型认知模型进行分页加载
- **图形缓存**：缓存已渲染的图形，提高切换速度
- **异步渲染**：在后台线程处理图形计算，避免阻塞UI
- **简化视图**：对超大型模型提供简化视图选项
