## 1. 组件设计原则

- **单一职责**：每个组件只负责一个明确的功能
- **可复用性**：设计通用组件，便于在不同页面复用
- **可组合性**：组件之间可以灵活组合，构建复杂页面
- **可测试性**：组件设计应便于单元测试和UI测试
- **性能优化**：考虑组件的性能，避免不必要的重渲染

## 2. 页面结构

### 2.1 主页面结构
- **顶部导航栏**：应用标题、设置按钮
- **中间内容区**：核心功能展示
- **底部操作栏**：语音输入按钮、主要功能入口

### 2.2 核心页面列表
1. **欢迎页面**
2. **登录/注册页面**
3. **主页（认知模型概览）**
4. **语音交互页面**
5. **多维度分析页面**
6. **认知模型详情页面**
7. **个性化设置页面**
8. **历史记录页面**

## 3. 核心组件设计

### 3.1 语音输入组件
- **样式**：圆形浮动按钮，位于屏幕底部中央
- **状态**：默认、录音中、录音完成
- **交互**：长按录音，松开停止
- **反馈**：录音时长显示、波形动画

### 3.2 认知模型可视化组件
- **类型**：力导向图、层次图、雷达图
- **交互**：缩放、拖拽、点击详情
- **样式**：节点颜色区分概念类型，边粗细表示关系强度

### 3.3 分析结果展示组件
- **类型**：雷达图、柱状图、饼图
- **数据更新**：实时更新，平滑过渡动画
- **交互**：点击查看详情，切换分析维度

## 4. 设计系统

### 4.1 色彩规范
| 颜色 | 用途 |
|------|------|
| 主色调（#4A90E2） | 按钮、导航栏、强调元素 |
| 辅助色（#50E3C2） | 成功状态、积极反馈 |
| 警告色（#F5A623） | 警告状态、需要注意的元素 |
| 错误色（#D0021B） | 错误状态、负面反馈 |
| 背景色（#F8F8F8） | 页面背景 |
| 卡片色（#FFFFFF） | 卡片、容器背景 |

### 4.2 字体规范
| 字号 | 用途 |
|------|------|
| 24pt | 页面标题 |
| 20pt | 卡片标题 |
| 16pt | 正文内容 |
| 14pt | 辅助文字 |
| 12pt | 提示文字 |

### 4.3 间距规范
| 间距 | 用途 |
|------|------|
| 8pt | 小间距（按钮内边距、文字间距） |
| 16pt | 中间距（卡片间距、区域间距） |
| 24pt | 大间距（页面边距、模块间距） |
| 32pt | 超大间距（主要内容区域） |
