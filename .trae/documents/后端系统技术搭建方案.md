# 后端系统技术搭建方案

## 1. 开发环境配置

### 1.1 安装Node.js和npm
- 确保安装Node.js 18+和npm
- 使用nvm或直接从官网下载安装

### 1.2 初始化项目
- 创建项目目录
- 初始化npm项目
- 安装TypeScript和相关依赖
- 配置tsconfig.json

### 1.3 安装核心依赖
- Fastify：Web框架
- SQLite：数据库
- Qdrant：向量数据库客户端
- TypeORM：ORM框架（可选，用于简化数据库操作）
- Zod：数据验证
- Pino：日志库

## 2. 项目基础结构搭建

按照Clean Architecture的分层设计，创建以下目录结构：

```
src/
├── domain/                # 领域层（纯净层）
│   ├── entities/         # 核心领域对象
│   ├── services/         # 领域服务
│   └── value-objects/    # 值对象
├── application/          # 应用层（用例层）
│   ├── usecases/         # 用例实现
│   ├── ports/            # 接口定义
│   └── dtos/             # 数据传输对象
├── infrastructure/       # 基础设施层
│   ├── persistence/      # 数据持久化
│   ├── ai/               # AI能力实现
│   └── event/            # 事件系统
├── interfaces/           # 接口层
│   └── http/             # HTTP API
│       ├── routes/       # 路由定义
│       └── server.ts     # 服务器配置
└── shared/               # 共享工具和常量
    ├── errors/           # 错误定义
    ├── utils/            # 工具函数
    └── constants.ts      # 常量定义
```

# 详细实施步骤

## 1. 初始化项目
```bash
mkdir cognitive-assistant
cd cognitive-assistant
npm init -y
npm install typescript ts-node tsup @types/node fastify sqlite3 qdrant-client zod pino
npm install --save-dev eslint prettier jest supertest
```

## 2. 配置TypeScript
```json
// tsconfig.json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "commonjs",
    "lib": ["ES2020"],
    "outDir": "./dist",
    "rootDir": "./src",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true,
    "resolveJsonModule": true,
    "declaration": true,
    "declarationMap": true,
    "sourceMap": true
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules", "dist"]
}
```

## 3. 创建项目目录结构
